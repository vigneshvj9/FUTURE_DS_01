# Customer Churn Analysis Dashboard

## Project Overview

This project analyzes customer churn behavior using Excel and Power BI. The objective of the analysis is to identify customer retention patterns, engagement behavior, and high-risk customer segments.

## Tools Used

- Microsoft Excel
- Power BI

## KPIs Analyzed

- Total Customers
- Churned Customers
- Average Lifetime Value
- Average Session Duration
- Login Frequency
- Mobile App Usage

## Key Insights

1. Customers with higher engagement levels are more likely to remain retained.
2. The 36–50 age group has the highest churn volume.
3. The USA contributes the largest customer and churn volume.
4. Customer lifetime value remains relatively consistent across countries.

## Business Recommendations

1. Improve customer engagement programs.
2. Focus retention strategies on high-risk customer segments.
3. Enhance user experience for low-engagement users.
